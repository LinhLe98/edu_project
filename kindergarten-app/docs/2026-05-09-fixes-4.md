### 2. About Page — 2-Column Intro Layout (Hero Left, School History Right)

**Root cause:** The About page had `app-page-hero` (full-width banner) followed by `app-school-history` (full-width timeline section), creating a tall, unbalanced layout.

**Fix:**
- Removed `PageHeroComponent` and `SchoolHistoryComponent` from `AboutComponent` imports.
- Inlined the school history `timeline[]` array directly into `AboutComponent` (6 entries).
- Replaced the two full-width sections with a single `<section>` containing a 2-column `lg:grid-cols-2` layout:
  - **Left column**: badge, `h1`, subtitle, 3-stat grid (15+ years, 350+ students, 25+ teachers), CTA buttons — uses `lg:sticky lg:top-24` to stay visible while scrolling the timeline.
  - **Right column**: school history timeline with lavender circle markers and card-base items.
- `ManagementBoardComponent`, `TeacherGroupsComponent`, `FacilitiesSectionComponent` remain unchanged below the intro section.

Files changed:
- `src/app/pages/about/about.component.ts` — removed PageHero/SchoolHistory imports, added timeline data
- `src/app/pages/about/about.component.html` — full rewrite of intro section; kept 3 sub-components below
- `src/app/pages/about/about.component.spec.ts` — updated: removed app-page-hero/app-school-history assertions; added timeline length and DOM rendering tests

### 4. News List — Search Box + Pagination

**New features in `NewsListComponent`:**

**Search box:**
- `<input>` above the category tabs with debounce-free `(input)` binding to `onSearch()`
- Filters by `title` and `excerpt` (case-insensitive contains)
- Resets `currentPage` to 1 on every query change

**Pagination:**
- `PAGE_SIZE = 6` articles per page
- `pagedArticles` computed signal slices `allFiltered` based on `currentPage`
- `totalPages` computed signal drives the page number buttons
- Previous/next chevron buttons, disabled at boundaries
- `goToPage()` calls `window.scrollTo({ top: 0, behavior: 'smooth' })` on navigation

Files changed:
- `src/app/pages/news/news-list.component.ts` — added `searchQuery`, `currentPage`, `allFiltered` (computed), `pagedArticles` (computed), `totalPages` (computed), `pageNumbers` (computed), `onSearch()`, `goToPage()`; added `FormsModule` import
- `src/app/pages/news/news-list.component.html` — added search input above tabs, changed grid to use `pagedArticles()`, added pagination row
- **NEW** `src/app/pages/news/news-list.component.spec.ts` — 8 tests covering initialization, category filter, search, page reset, pagination slicing, and `totalPages`
