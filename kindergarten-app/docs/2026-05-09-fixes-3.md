# 2026-05-09 — Feature Fixes: Navigation, Engagement & Teacher Groups

## Summary
Four feature fixes and improvements implemented. All changes pass `ng build` with zero errors.
Unit tests added or updated for every modified component.

## Changes

### 1. Teacher Highlights → now scrolls to correct About section

**Root cause:** The "Xem tất cả giáo viên" button in `latest-news.component.html` linked to
`/gioi-thieu` without a fragment, landing at the top of the About page (school history)
rather than the teacher groups section.

**Fix:** Added `fragment="teacher-groups"` to the `routerLink` anchor. The About page already
has `id="teacher-groups"` on the teacher-groups section, so Angular Router now scrolls there
directly via `withInMemoryScrolling({ anchorScrolling: 'enabled' })`.

Files changed:
- `src/app/pages/home/teacher-highlights/teacher-highlights.component.html`
- `src/app/pages/home/teacher-highlights/teacher-highlights.component.spec.ts` — added test verifying `href="/gioi-thieu#teacher-groups"`

### 2. News Detail — Emoji Reactions & Comments Section

**New features added to `NewsDetailComponent`:**

**Reactions bar:**
- 4 emoji reactions (❤️ 👍 😊 🎉) rendered as toggle buttons
- Each click increments/decrements the count and highlights the button
- Counts and reacted state reset when navigating to a different article

**Comments section:**
- Existing comments list with avatar initial, name, and date
- Comment form with name + text fields (`[(ngModel)]`, `FormsModule`)
- `submitComment()` validates that both fields are non-empty before adding
- Temporary success notification shown for 3 seconds after submit
- Comments and reactions reset on article navigation (inside `paramMap` subscriber)

Files changed:
- `src/app/pages/news/news-detail.component.ts` — added `NewsReaction`, `NewsComment` interfaces; `reactions`, `comments` signals; `react()`, `submitComment()` methods; `FormsModule` import
- `src/app/pages/news/news-detail.component.html` — added reactions bar and comments sections
- `src/app/pages/news/news-detail.component.spec.ts` — added 4 new tests (total: 10) covering initial reaction state, toggle, comment submit, and empty-field guard

### 3. Contact Page — Support Request Form

**Root cause:** The contact page only showed a "call us" CTA card with no way for parents to
submit a written request.

**Fix:** Replaced the CTA card with a full support request form:
- Fields: Họ và tên (required), Số điện thoại, Email, Chủ đề (select), Nội dung (required)
- `submitForm()` validates name + message; sets `formError` signal if blank, `formSuccess` if valid
- On success: shows a green confirmation banner and hides the form
- Map embed moved to right column of the contact info section; form is a separate full-width section

Files changed:
- `src/app/pages/contact/contact.component.ts` — added `SupportRequest` interface; `formData`, `formSuccess`, `formError` signals; `subjectOptions` array; `submitForm()` method; `FormsModule`, `NgIf` imports
- `src/app/pages/contact/contact.component.html` — restructured layout; added form section
- `src/app/pages/contact/contact.component.spec.ts` — added 3 new tests (total: 6) covering form initial state, valid submit, and empty-field error

### 4. Teacher Groups — Per-Group "See All" Pages

**Feature:** Each teacher group on the About page now has an individual "Xem tất cả →" link
routing to `/to-giao-vien/:dept`, which shows the full roster for that specific group.

**Changes:**
- `teacher-groups.component.ts` — added `dept: StaffDepartment` field to each group object
- `teacher-groups.component.html` — replaced the single global "Xem thêm" link at the bottom
  with a per-group "Xem tất cả →" link in the group header row
- **NEW** `src/app/pages/teacher-group/teacher-group-detail.component.ts` — standalone component
  subscribing to `:dept` paramMap, fetching all staff via `StaffService.getByDepartment()`,
  mapping dept key to Vietnamese label + subtitle via `GROUP_META` lookup table
- **NEW** `src/app/pages/teacher-group/teacher-group-detail.component.html` — page hero +
  4-column teacher grid + back link to `/gioi-thieu#teacher-groups`
- **NEW** `src/app/pages/teacher-group/teacher-group-detail.component.spec.ts` — 5 tests
  covering init, member loading, label mapping, card rendering, and back link
- `src/app/app.routes.ts` — added `/to-giao-vien/:dept` lazy route
- `src/app/pages/about/teacher-groups/teacher-groups.component.spec.ts` — updated mock to
  include `getGroupLeader`/`getGroupViceLeaders`; added `provideRouter([])`; added link test
